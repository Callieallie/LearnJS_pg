HTML and CSS Notes

HTML Elements (they become tags when wrapped in <>)

  • BLOCK ELEMENTS
    • h1: The most important topic of the page (Big Newspaper Title) ONE PER PAGE
    • h2: Defines titles of the page's major sections (sub-sections of h1)
    • h3-h6: Each heading-type represents a further subsection of it's predecessor
    • ***Think of headers like you would the alphanumeric outline system in paper drafting
    • ***Roman Numerals, Capitalized Letters, Arabic Numerals (1-9), and Lowercase Letters
    • br: used to add a newline/carriage-return. NOT FOR SEPERATING PARAGRAPHS (or elements)!
      1. The br element should be reserved for poems and addresses.
      2. DO NOT use the br element to format a paragraph.
  • LANDMARK ELEMENTS
    1. main: Represents the dominant content of the page
    2. header: Contains any repeated Intro to the Page content (often the h1)
    3. footer: Contains miscelanious information towards the end of a page
  • INLINE ELEMENTS
    1. strong: used to strongly emphasize important sections of text in a paragraph. Is inline in nature.
      1. DO NOT use a strong element as a way to bolden text.
      2. HTML should be used for meaning. If you need bold text, use CSS.
      3. Use strong to point out words or phrases that encompass the meaning of a paragraph or page.
    2. b: "bring attention to". Draws attention but does not add emphasis.
      1. Use b to add some contrast in focus.
      2. Imagine the text in the b element saying "notice me :)"
    3. em: "emphasis". Does not have to do with importance, but rather inflection.
      1. "Did *you* find the keys?" emphasizes "you"
      2. Again, DO NOT use em or any other HTML element for style.
    4. i: "idiomatic text". Word(s) spoken in a differing voice or mood.
      1. Often used for parts of text in a different language.
      2. Can be used for the voice of a character in a story, or to show sarcasm.
      3. The lang attribute should be put on any i element representing a language change.

Assorted Notes

  1. emmet: a plugin used to assist in writing HTML quickly. Already installed with VS Code
  2. Meaningful headings are important especially for screen readers and for SEO
  3. Axe is the name of a popular accessability testing tool
  4. Boilerplate: "!"+tab will create a basic HTML template
  5. COMMENTS: Ctrl+"/" will create a new html comment
  6. Validating HTML: validator.w3.org
