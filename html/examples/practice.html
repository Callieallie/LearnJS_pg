Headers

  1. This is the precedance an h1 element should have. There can only be one h1 per page.
    1. An h2 element should be a more narrowed topic pertaining to the h1.
    2. You can have as many h2-h6 elements as you would like.
      1. h3 elements are intended for sub-topics of h2 elements.
      2. Think of h1 as the title of a book, h2 as the chapter title, and h3 as the topic of a section of the chapter.
        1. If h3 elements have sub-sections, implement them with h4 elements.

Paragraphs

It is important to use paragraphs as you would in an English class. Each paragraph should have its own topic. A paragraph element should not hold several paragraphs of content. Instead, stick to using a seperate paragraph element for each paragraph of content.

A paragraph can be as short as a single sentence or prompt or as long as you want so long as you stick to the one topic.

Although you may be tempted to add newline breaks with the br element, it is not reccomended that you do so unless you are working with an address or a poem.

Lists: Ordered (ol) and Unordered (ul)

Ordered and unordered lists are commonly reffered to as bullet or numbered lists in document editing software such as Microsoft Word or Google Docs.

An ordered list will indicate some sort of precidence in the list of each list item. Think of the classic college paper outline. In an outline, it matters which topic is first, second, third, etc.

An unordered list is comparable to the standard bullet list. In an unordered list, it does not matter which list element is listed first, second, or third.

The following displays the general format of a list.

<ul>
  <li>Item One</li>
  <li>Item Two</li>
  <li>Item Three</li>
</ul>

Lists can also be nested inside of one another. When nesting lists, be careful to include the nested list in the list item element it is a part of.

<ul>
  <li>Item One</li>
  <li>Item Two</li>
  <li>Item Three
    <ul>
      <li>Nested Item One</li>
      <li>Nested Item Two</li>
    </ul>
  </li>
</ul>
